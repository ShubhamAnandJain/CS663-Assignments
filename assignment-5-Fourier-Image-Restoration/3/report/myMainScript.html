To remove the periodic noise, we apply a notch filter. The steps are given below:

  1. Pad the HxW image to get a 2Hx2W image.
  2. Calculate the 2D Fourier Transform.
  3. Examine the log magnitude plot to find the noise frequency. Say bin (x, y) corresponds to the noise.
  4. Apply a notch filter centred at (x,y). The notch filter has a one sided-window of length 7 such that the border pixels have an attenuation of and as we move in towards the centre, the attenuation of all 8 pixels which are equidistant from the centre gets multiplied by .
  5. Since the image is real, the plot is symmetric about the centre frequency. hence, we need to apply the same filter to the pixel which is symmetrically located from (x,y), across the centre bin.
  6. Calculate the inverse 2D DFT and crop out the central HxW portion. This is the final desired filtered image.
